# Experiment 49: 'this' Keyword

## Problem Statement

Write a program to demonstrate the use of the `this` keyword to distinguish between class attributes and method parameters with the same name.

Create a class `ThisDemo` with:
1. An integer instance variable named `value`.
2. A method `setValue(int value)` that uses `this` to assign the parameter to the instance variable.
3. A method `display()` to print the value.

## Input Format

* An integer value.

## Output Format

* `Value: [value]`

### Example 1

**Input:**

```text
50
```

**Output:**

```text
Value: 50
```
